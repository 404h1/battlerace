# Ported BattleRace Features

These features were moved into `race.html`.

## Map Themes

Use the in-game map selector on the top right, or open with a URL:

```text
/?theme=busan
/?theme=mountain
/?theme=valley
```

## Game Over Message

```text
/?gameOverText=으이그%20바보!
/?ment=다시%20해봐!
```

## Jump Item

Green boxes charge jumps. Press `Space` to use a charged jump.

```text
/?jumpMs=5000
/?jumpCharges=5
```

## Easter Eggs

Type these while the game is focused:

```text
RAINBOW
JJUMP
BOSAN
```
